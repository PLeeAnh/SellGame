## Hướng dẫn cài đặt
1. Mở GitHub Desktop
2. Click **File → Clone repository**
3. Chọn tab **GitHub.com**
4. Local path: Chọn nơi file cần lưu nhớ là file phải empty
5. Click **Clone**
